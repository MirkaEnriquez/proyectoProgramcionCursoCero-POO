# Sistema de Reservas de Restaurante en C++

## Descripción
Este proyecto consiste en un sistema desarrollado en C++ que permite gestionar las reservas de un restaurante de manera sencilla.
El programa utiliza arreglos y funciones para registrar, buscar, cancelar y visualizar reservas, así como para consultar la disponibilidad de mesas en diferentes horarios.

## Funcionalidades principales
- Registrar una reserva (nombre, horario y tipo de mesa)
- Mostrar todas las reservas activas
- Buscar una reserva por nombre
- Cancelar una reserva existente
- Ver la disponibilidad de mesas según horario y tipo

## Estructura básica del proyecto
- main.cpp: contiene la lógica principal del programa

## Compilación y ejecución
Para compilar el programa, usar un compilador de C++ como g++:
- g++ programa.cpp -o reservas

Para ejecutar el programa:
- ./reservas

- reservas.exe
